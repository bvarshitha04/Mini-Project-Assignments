# Personal Expense Tracker 📊

Hey there! This is a Python script I wrote to help analyze monthly spending. Looking at raw data is boring, so this script turns an Excel/CSV file of expenses into a visual chart!

### What it does:
- **Reads Data:** It reads all your spending history from `expenses.csv`.
- **Categorizes Spending:** It groups the expenses up (like Food, Rent, Entertainment).
- **Draws a Chart:** It automatically generates a visual graph (`output.png`) so you can actually see where your money is going!

### How I built it:
- **Language:** Python
- **Tools Used:** I used popular data science libraries (like Pandas for handling the CSV data, and Matplotlib/Seaborn for drawing the chart).
