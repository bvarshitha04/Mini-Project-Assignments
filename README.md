# Mini-Project Assignments 🚀

Welcome to my mini-projects repository! This repository contains a collection of 7 different mini-projects I built using **Java**, **Python**, and **SQL**. 

These projects showcase my ability to build robust backend systems, analyze large sets of data, and create automation scripts.

## ☕ Java Projects
1. **[Library Management System](./java/Library-Management-System/)**: A fully automated console application built with Core Java and OOP principles to manage book inventory, user borrowing, and automatic fine calculation.
2. **[FinSafe Transaction System](./java/FinSafe-Transaction-System/)**: A secure banking simulation that handles deposits and withdrawals while safely catching errors (like insufficient funds) using Custom Exceptions.

## 🐍 Python Projects
3. **[Personal Expense Tracker](./python/Expense-Tracker/)**: A data analysis script that reads raw CSV expense data and automatically generates visual charts to track monthly spending.
4. **[OpsBot Log Analyzer](./python/OpsBot-Log-Analyzer/)**: An automation bot that instantly scans massive server log files to hunt down security alerts and critical errors using Regular Expressions.

## 🗄️ SQL Projects
5. **[Retail Insights Analytics](./sql/Retail-Insights/)**: A Data Analyst project for a supermarket chain. Uses complex queries to identify dead stock, track expiring products, and calculate monthly revenue.
6. **[Movie Recommendation System](./sql/Movie-Recommendation/)**: The backend database logic for a streaming platform that tracks watch history and recommends new movies based on user viewing patterns.
7. **[Digital Library Database](./sql/Digital-Library/)**: A structural blueprint (DDL) and mock data system (DML) designed to securely store and link library members, books, and loan records.
