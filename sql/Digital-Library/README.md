# Digital Library Database 📖

Welcome to my Digital Library SQL project! This script handles the backend database setup for a local library, allowing librarians to track books, members, and book loans perfectly.

### What it does:
- **Book & Member Storage:** Creates safe, structured tables for storing all the books in the library and all registered members.
- **Issue Tracking:** Keeps a permanent record of which member borrowed which book and when it needs to be returned.

### How I built it:
- **Language:** SQL
- **Concepts Used:** This focuses on DDL (Data Definition Language) to build the structure with Primary Keys and Foreign Keys, ensuring that you can't loan a book to a user that doesn't exist! I also used DML (Data Manipulation Language) to insert test data.
