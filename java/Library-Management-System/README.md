# Library Management System 📚

Hey! This is a simple Java project I built to automate a library's manual work. Instead of using pen and paper to track who borrowed what, this console app does it all.

### What it does:
- **Register Users & Books:** Easily add new books and library members.
- **Track Borrowing:** You can issue a book to a user and the system remembers it.
- **Smart Fines:** It automatically calculates a $5 fine per day if someone returns a book late!
- **Search:** You can look up books by their title or author.

### How I built it:
- **Language:** Pure Core Java.
- **OOP Concepts:** I used Encapsulation (keeping variables private and safe) and Abstraction (hiding the messy logic in the background).
- **Date Tracking:** I used Java's built-in `LocalDate` to figure out exactly how many days late a book is.
